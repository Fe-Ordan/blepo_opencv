Blepo Demo

Screenshot:

Download and run the demo (Windows executable).

The resizable figure window, shown below, displays the image along with individual pixel values. The menu allows the user to load or save PGM/PPM/BMP/JPEG files, and to write EPS files (especially handy for including figures in Latex documents). Mouse clicks can be captured effortlessly using methods of the figure class, without having to specify callbacks.
